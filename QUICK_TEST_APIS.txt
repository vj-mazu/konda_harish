## TEST WITH CURL:

```bash
# 1. Login
curl -X POST http://localhost:5000/api/auth/login -H "Content-Type: application/json" -d "{\"username\":\"admin\",\"password\":\"password\"}"

# 2. Test arrivals (replace TOKEN)
curl -X GET http://localhost:5000/api/arrivals -H "Authorization: Bearer TOKEN"

# 3. Test dashboard
curl -X GET http://localhost:5000/api/dashboard -H "Authorization: Bearer TOKEN"

# 4. Test stock
curl -X GET http://localhost:5000/api/records/stock -H "Authorization: Bearer TOKEN"

```
